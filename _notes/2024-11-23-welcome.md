# 欢迎使用技术笔记博客

这是您的第一篇技术笔记！本博客支持 Markdown 语法，让您可以轻松编写格式化的技术文档。

## Markdown 基础语法

### 标题
```markdown
# 一级标题
## 二级标题
### 三级标题
```

### 列表
- 无序列表项
- 另一个列表项

1. 有序列表项
2. 第二个列表项

### 代码块

支持多种编程语言的代码高亮：

```python
def hello_world():
    print("Hello, World!")
    return True
```

```javascript
function greet(name) {
    console.log(`Hello, ${name}!`);
    return `Welcome, ${name}`;
}
```

$$

$$

## 使用提示

1. 文件名格式：`YYYY-MM-DD-标题.md`
2. 在文件开头添加 YAML front matter
3. 使用标签对笔记进行分类
4. 定期备份您的笔记

开始编写您的技术笔记吧！
